# 🛒 Grocery List

## 📌 Project Description

Grocery List is a simple shopping list app created with React.

This app helps users organize their shopping tasks. Users can add items, mark them as completed, and delete items when they are no longer needed.

I created this project to practice React, components, and working with user actions.

## ⚙️ Installation

To run this project locally, follow these steps:

```bash
git clone https://github.com/OlgaVetrov/GroceryList.git

cd GroceryList

npm install

npm start
```

After running the command, the application will open in your browser.

## 🚀 Usage

🛒 **Grocery List helps you organize your shopping tasks easily.**

With this app, you can create your shopping list and manage items step by step.

### ✨ What you can do:

- ➕ Add new grocery items
- ✅ Mark items as completed (crossed)
- 🗑️ Delete items you no longer need
- 📝 Keep your shopping list organized
